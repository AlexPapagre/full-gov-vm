My City Gov
===

---

Run the application:
```shell
cd my-city-gov
./mvnw spring-boot:run
```

---

Start minIO server:
```shell
cd my-city-gov
./start-minio.sh
```

---

Stop minIO server:
```shell
cd my-city-gov
./stop-minio.sh
```

---

Open in browser: [localhost:8080](http://localhost:8080)

---

Open minIO in browser: [localhost:9000](http://localhost:9000)

---
